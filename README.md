# Data Structures and Algorithms (DSA) with C/C++

This repository contains implementations of fundamental Data Structures and Algorithms (DSA) using C and C++. It is designed for learning and practicing DSA concepts.

## Directory Structure

- **STACK**: Implementations of stack data structures.  
- **QUEUE**: Implementations of queue data structures.  
- **LINKED LIST**: Singly and doubly linked list implementations.  
- **RECURSION**: Examples and exercises on recursion.  
- **TREE**: Binary trees and related algorithms.  
- **SORTING**: Various sorting algorithms.  
- **SEARCHING**: Implementations of searching algorithms.
